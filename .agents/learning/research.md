# Research Rules

## Next.js research

- Before writing framework-specific code, read the relevant official docs shipped in `node_modules/next/dist/docs/`.
- Prefer local docs over memory because this project uses a newer version with breaking changes.

## Reference-site research

- The project may use Coolmate as a visual/product-flow reference.
- Do not scrape protected pages, bypass anti-bot protections, or copy private data.
- If the user provides screenshots or authorized assets, use those as references.
- If browsing public pages, summarize patterns and recreate them with original code and assets.

## Skill research

The user referenced `https://github.com/google-labs-code/stitch-skills` as inspiration for frontend and brainstorming skills. Use the idea of reusable, task-specific skill prompts, but keep this repo's `.agents` files tailored to the Coolmate clone and do not depend on that external repo being installed.
